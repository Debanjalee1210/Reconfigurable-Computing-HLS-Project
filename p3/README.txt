README FILE FOR PART 3

To reproduce the results,the follwoing steps need to be followed:-
Step 1: Create a solution as default name "Solution 1"from Project->New Solution after following the README for Part 2.
Step 2: Click on the created Solution1 in the explorer window.
Step 2: Replace the directive.tcl inside the Solution1->constraints folder
Step 3: Set the Solution1 as "Set Active Solution" by right clicking on it and run "C synthesis"->"Active Solution" by right clicking on it.
Step 4: To verify the results , click on Solution1->syn->report->csynth.rpt to view the synthesis report 
Step 5: Follow step 1 to Step 4 for all other solutions.
